Header file containing structures and classes related to graph representation.

This file defines structures and classes representing a graph data structure. The 'Vertex' class represents vertices in the graph, containing information about the vertex, including its adjacency list and other properties. The 'Edge' class represents edges between vertices, containing information about the edge, including the destination vertex and weight. The 'Graph' class represents the entire graph, encapsulating the collection of vertices and edges.
